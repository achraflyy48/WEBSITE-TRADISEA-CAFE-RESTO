# TradiSea Website

Website bisnis resmi TradiSea — Cafe & Resto, Penginapan, Kebun, dan Keberlanjutan di Situbondo, Jawa Timur.

## Struktur File

```
tradisea-website/
├── index.html      # Halaman utama (one-page)
├── style.css       # Seluruh styling
├── script.js       # Interaksi + multi-bahasa
├── assets/
│   ├── images/     # Tempatkan foto-foto di sini
│   └── icons/
└── README.md
```

## Fitur

- ✅ One-page website dengan anchor navigation
- ✅ Toggle bahasa Indonesia / English
- ✅ Sticky navbar dengan active state
- ✅ Mobile-responsive (hamburger menu)
- ✅ Scroll reveal animations
- ✅ Tombol WhatsApp & Email langsung
- ✅ Embed peta Situbondo
- ✅ Wave dividers sebagai elemen visual signature

## Cara Menambahkan Foto

Di `style.css`, ganti class placeholder:

```css
.resto-placeholder { background: url('assets/images/resto.jpg') center/cover; }
.kamar-placeholder { background: url('assets/images/kamar.jpg') center/cover; }
.kebun-placeholder { background: url('assets/images/kebun.jpg') center/cover; }
```

## Deploy

Website ini statis — cukup upload semua file ke hosting (Netlify, Vercel, cPanel, dll).
